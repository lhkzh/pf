**专注fibjs接口开发的框架**    
 -
**特性**    
 - typescript编写  
 - 注解配置路由、参数规则、输出格式、权限判断  
 - 自动生成简单的在线接口文档（需要按照规则写注释    
 - fibjs纤程的js接口编写体验（无callback、无promise）
 - 支持热更新    
 - 不建议使用cookie和session这种授权体系
